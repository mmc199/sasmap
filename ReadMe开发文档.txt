z 坐标是反着的怎么添加啊
就是第一级，z=18
z变量支持：{$z}{$z+n}{$z-n}{$z|n} ，你可以用{$z|n} = n- z
x变量：{$x}{$x+n}{$x-n}{$x/n}，其中n为数值，运算符号+-/分别代表加、减、除
y变量：{$y}{$y+n}{$y-n}{$y/n}{$y!}{$y@}，{$y!} = 2的z次方-1-y，{$y@} = 2的z-1次方-1-y
